# geojson2poly
Convert geojson file to poly file

#### What is geojson?
http://geojson.org/


#### What is poly
https://wiki.openstreetmap.org/wiki/Osmosis/Polygon_Filter_File_Format


#### Build
```shell
$ go build -o geojson2poly main.go
```


#### Usage
```shell
$ geojson2poly -i infile.json -o outfile.poly
```
